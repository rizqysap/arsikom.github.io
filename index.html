ARSITEKTUR KOMPUTER

Arsitektur Komputer merupakan ilmu yang mempelajari struktur internal suatu komputer beserta bagaimana komponen-komponennya saling berhubungan dan bekerja sama dalam menjalankan tugas-tugasnya.

CENTRAL PROCESING UNIT

Unit Pemrosesan Pusat (CPU) adalah otak dari sebuah komputer. CPU merujuk pada bagian dari sebuah komputer yang bertanggung jawab untuk menjalankan tugas komputasi. Komponen-komponen CPU meliputi ALU, CU, Register, Cache, dan bus.

MEMORY

Memory adalah salah satu komponen penting dalam komputer yang berfungsi untuk menyimpan dan mengakses data dengan cepat.Jenis memory yaitu RAM (Random Access Memory), ROM (Read Only Memory), Chace.

DATA NON - NUMERIK

Pada dunia komputasi, data non numerik merujuk pada jenis data yang bukan berupa angka atau bilangan. Jenis-jenis data non numerik dapat mencakup teks, gambar, suara, video, dan banyak lagi.

DASAR BAHASA PEMROGRAMAN

Belajar bahasa pemrograman tidak hanya memungkinkan kamu untuk membuat aplikasi dan website yang berguna, tapi juga dapat mengasah kreativitasmu.

COMPILER, INTERPRETER, DAN LINKER

Compiler, Interpreter, dan Linker adalah komponen penting dalam pengembangan perangkat lunak. Di sini kita akan mempelajari pengertian, fungsi, cara kerja, perbedaan, dan manfaat masing-masing komponen tersebut.
